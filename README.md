## snippetbox
##### this application lets people paste and share snippets of text — a bit like Pastebin or GitHub’s Gists. 

#### To run the app:

1. Clone the project to your local machine:
    git clone https://www.github.com/MlondiMchunu/snippetbox.git

2. Run the app:
    go run ./cmd/web
